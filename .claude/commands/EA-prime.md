# Prime Skills Module

## Purpose

Get a quick overview of this module's structure, included skills, and available guides. Use this when starting a session to understand what's available.

## Workflow

1. Read the README.md for module overview

2. List available skills:
   - Glob for SKILL.md files in skills/
   - Extract name and description from each

3. List available guides:
   - Glob for .md files in guides/
   - Present as a reference list

4. Note the showcase folder contents:
   - Show what File Factory can produce

5. Summarize key concepts from the module

## Report

```
## Skills Module Overview

### Available Skills
- skill-name: brief description
...

### Guides
- guide-name.md: topic covered
...

### Showcase
- file-factory-output/: Example documents created with File Factory

### Quick Start
1. Run /EA-install to add skills
2. Read guides/ for understanding
3. Explore showcase/ for examples
```

## Instructions

- Keep output concise - bullet points preferred
- Focus on what's actionable
- Highlight the 5 included skills
- Mention the 21,000+ external skills available in resources.md
