# Art Deco Gold Theme

**Character**: Luxury, timeless elegance, 1920s glamour

## Color Palette
- Midnight: `#0A0020` - Deep background (RGB: 10, 0, 32)
- Gold: `#D4AF37` - Primary metallic accent (RGB: 212, 175, 55)
- Beige: `#F5F5DC` - Soft neutral text (RGB: 245, 245, 220)
- Black: `#000000` - Strong contrast (RGB: 0, 0, 0)

## Typography
- Headers: Georgia, Times New Roman (serif)
- Body: Georgia, Arial

## Best For
Luxury brands, historical content, awards ceremonies, high-end events

## Effects
- Golden glow (blurRad: 76200, color: D4AF37)
- Thin gold divider lines
- Numbered badges in gold circles
- Roman numeral accents

## CSS Example
```css
.art-deco-gold {
  background: #0A0020;
  color: #F5F5DC;
}
.gold-accent {
  color: #D4AF37;
  text-shadow: 0 0 8px rgba(212, 175, 55, 0.6);
}
.gold-badge {
  background: #D4AF37;
  color: #0A0020;
  border-radius: 50%;
}
```
