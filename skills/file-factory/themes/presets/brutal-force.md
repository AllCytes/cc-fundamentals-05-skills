# Brutal Force Theme

**Character**: Raw, powerful, industrial

## Color Palette
- Dark Gray: `#1A1A1A` - Concrete-like background (RGB: 26, 26, 26)
- Magenta: `#FF00FF` - Neon primary (RGB: 255, 0, 255)
- Cyan: `#00FFFF` - Neon secondary (RGB: 0, 255, 255)
- Orange: `#FF6600` - Warning/accent (RGB: 255, 102, 0)

## Typography
- Headers: Impact (bold, aggressive)
- Body: Arial, Courier New (technical)

## Best For
Construction, manufacturing, industrial tech, urban brands, punk aesthetic

## Effects
- Thick neon borders (38100 EMUs = 3pt)
- Semi-transparent black overlays (alpha: 70000)
- Colored accent bars/stripes
- High contrast borders

## CSS Example
```css
.brutal-force {
  background: #1A1A1A;
  color: #FFFFFF;
}
.brutal-primary {
  color: #FF00FF;
  border: 3pt solid #FF00FF;
}
.brutal-secondary {
  color: #00FFFF;
  border: 3pt solid #00FFFF;
}
.brutal-warning { color: #FF6600; }
```
