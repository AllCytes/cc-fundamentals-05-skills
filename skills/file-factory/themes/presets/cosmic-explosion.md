# Cosmic Explosion Theme

**Character**: Energetic, explosive, vibrant

## Color Palette
- Dark Space: `#1A0030` - Deep purple background (RGB: 26, 0, 48)
- Solar Orange: `#FF6B35` - Warm primary accent (RGB: 255, 107, 53)
- Electric Blue: `#00D4FF` - Cool contrast accent (RGB: 0, 212, 255)
- Gold: `#FFD700` - Highlight color (RGB: 255, 215, 0)

## Typography
- Headers: Impact, Arial Black
- Body: Arial, Tahoma

## Best For
Science presentations, space themes, explosive product launches, energy brands

## Effects
- Radial gradient backgrounds (center burst)
- Starburst patterns
- Multi-color glow effects

## CSS Example
```css
.cosmic-explosion {
  background: radial-gradient(circle, #FF6B35 0%, #1A0030 100%);
  color: #FFFFFF;
}
.cosmic-primary { color: #FF6B35; }
.cosmic-secondary { color: #00D4FF; }
.cosmic-accent { color: #FFD700; }
```
