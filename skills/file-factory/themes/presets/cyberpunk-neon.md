# Cyberpunk Neon

A bold, futuristic theme with electric neon colors on dark backgrounds. Perfect for tech, AI, and futuristic content.

## Color Palette

- **Neon Blue**: `#00D4FF` - Primary electric accent
- **Electric Purple**: `#8B5CF6` - Secondary vibrant accent
- **Cyber Pink**: `#EC4899` - Highlight and emphasis
- **Matrix Green**: `#10B981` - Data and code elements
- **Deep Black**: `#0A0A0F` - Primary background
- **Gold**: `#FBBF24` - Premium accents
- **Silver**: `#94A3B8` - Secondary text
- **White**: `#FFFFFF` - Primary text on dark

## Typography

- **Headers**: Impact, Arial Black, or Segoe UI Bold
- **Body Text**: Segoe UI, Arial
- **Code/Data**: Consolas, Courier New

## Recommended Complexity

Level 5-6 (Extreme to Ultra-Extreme)

## Design Elements

- Dark backgrounds with neon text
- Glow border effects (4pt colored borders)
- Matrix-style decorative patterns (◈◈◈◈◈◈◈)
- Cyber symbols and icons
- Terminal/code-style text blocks
- Stats bars with emoji icons

## Best Used For

AI/ML presentations, tech startups, gaming content, futuristic products, developer documentation, innovation showcases.
