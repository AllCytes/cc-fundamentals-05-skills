# Neon Cyberpunk Theme

**Character**: Electric, futuristic, high-impact

## Color Palette
- Deep Void: `#0A0020` - Primary dark background (RGB: 10, 0, 32)
- Magenta: `#FF00FF` - Neon primary accent (RGB: 255, 0, 255)
- Cyan: `#00FFFF` - Neon secondary accent (RGB: 0, 255, 255)
- Neon Green: `#39FF14` - Highlight accent (RGB: 57, 255, 20)

## Typography
- Headers: Impact, Arial Black, or Arial Bold
- Body: Arial, Verdana

## Best For
Futuristic presentations, gaming, tech showcases, music events, nightlife

## Effects
- Intense neon glow (blurRad: 127000, alpha: 80%)
- Semi-transparent overlays (40-80% opacity)
- High contrast color combinations

## CSS Example
```css
.neon-cyberpunk {
  background: #0A0020;
  color: #FFFFFF;
}
.neon-primary { color: #FF00FF; text-shadow: 0 0 10px #FF00FF; }
.neon-secondary { color: #00FFFF; text-shadow: 0 0 10px #00FFFF; }
.neon-accent { color: #39FF14; text-shadow: 0 0 10px #39FF14; }
```
