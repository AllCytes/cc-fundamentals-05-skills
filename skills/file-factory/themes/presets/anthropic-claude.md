# Anthropic Claude

A warm, professional theme based on Anthropic's brand identity. Features the signature coral/orange palette with developer-friendly accents.

## Color Palette

- **Anthropic Orange**: `#DA7756` - Primary brand color
- **Dark Orange**: `#C4613C` - Darker accent
- **Light Orange**: `#F5D0C0` - Backgrounds and highlights
- **Peach Cream**: `#FEF3EF` - Light backgrounds
- **Terminal Blue**: `#3B82F6` - Code and links
- **Code Green**: `#10B981` - Success and code syntax
- **Charcoal**: `#1F2937` - Primary text
- **Slate**: `#475569` - Secondary text
- **White**: `#FFFFFF` - Clean backgrounds

## Typography

- **Headers**: Arial, Segoe UI Bold
- **Body Text**: Arial, Segoe UI
- **Code/Data**: Consolas, Monaco, Courier New

## Recommended Complexity

Level 3-6 (Enhanced to Ultra-Extreme)

## Design Elements

- Warm coral/orange accents on light backgrounds
- Clean, professional layouts
- Code blocks with dark backgrounds
- Command-line style elements
- Developer-friendly aesthetics
- Subtle gradients in extreme versions

## Best Used For

AI documentation, developer guides, Claude Code tutorials, Anthropic-related content, API documentation, technical specifications.

## Brand Guidelines

- Use orange sparingly as accent, not overwhelming
- Maintain clean, readable layouts
- Code examples should use dark terminal backgrounds
- Professional but approachable tone
