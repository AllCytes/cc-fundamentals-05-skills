What Gets Stored

Daily Wins

Three wins captured each evening: Relationships, Personal growth, and Health/wellness achievements.

Productivity Scores

0-100 score across 5 categories: Learning (25), Building (25), Organization (20), Social (15), Health (15).

Tomorrow's Priorities

JSON objects with task, time_window, category, difficulty, and Eisenhower quadrant for each focus area.

Predictions

Satisfaction predictions and burnout risk assessments that improve accuracy over time.
