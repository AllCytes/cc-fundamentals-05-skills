Self-Improving Patterns

A

ACT

Commands execute useful actions: /morning generates priorities, /reflect scores productivity, /delegate identifies automation opportunities.

L

LEARN

Store insights for future use. Track satisfaction prediction accuracy, record task batching patterns, note recurring themes and correlations.

R

REUSE

Apply learnings to future sessions. Adjust thresholds based on history, recommend proven task combinations, warn about burnout patterns.

Patterns Discovered Over Time

High Productivity Days

Morning exercise + 2-3 Q2 tasks + limited meetings = peak performance

Burnout Indicators

Multiple Q1 tasks + 5+ consecutive high days = risk warning

Optimal Batching

Learning before building + social after deep work = higher satisfaction

Prediction Accuracy

System tracks predicted vs actual satisfaction to improve recommendations
