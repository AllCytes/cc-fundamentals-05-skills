Daily Workflow

8-10 AM

Morning: /morning

Start the day with clarity on what matters most.

  • Review yesterday's planned focus
  • See Eisenhower Matrix view
  • Confirm top 3 priorities
  • Store in memory for afternoon

12-3 PM

Afternoon: /update

Quick status check to stay on track.

  • Status on each priority
  • Update TickTick tasks
  • Check for new urgent items
  • Adjust plan if needed

9 PM

Evening: /reflect

Capture wins and plan tomorrow.

  • Record 3 wins by category
  • Score productivity (0-100)
  • Generate tomorrow's priorities
  • Write to Supabase

~30 min/day

Total daily investment for complete workflow. System learns and improves over time.
